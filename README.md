# Customer Shopping Behavior Analytics Dashboard

An end-to-end retail analytics project that transforms raw customer shopping data into actionable business insights using Python, PostgreSQL, SQL, and Power BI. The project focuses on customer segmentation, purchasing behavior, product performance, and sales analytics through an interactive business intelligence dashboard.

---

## Project Overview

This project demonstrates the complete analytics workflow—from data preprocessing and database management to SQL-based business analysis and interactive dashboard development. The objective is to uncover customer purchasing patterns and provide insights that support data-driven business decisions.

---

## Tech Stack

- **Python** (Pandas, NumPy)
- **PostgreSQL**
- **SQL**
- **Power BI**
- **Jupyter Notebook**

---

## Project Workflow

### 1. Data Preparation
- Cleaned and preprocessed raw customer shopping data using Python.
- Handled missing values and standardized categorical variables.
- Performed feature engineering to improve downstream analysis.
- Exported the processed dataset for database integration.

### 2. Database Management
- Imported the cleaned dataset into PostgreSQL.
- Designed SQL tables for efficient querying and reporting.
- Validated data integrity and optimized the dataset for analysis.

### 3. Business Analysis
Performed SQL analysis to answer key business questions, including:

- Customer demographics
- Revenue by product category
- Shopping frequency analysis
- Purchase trends
- Product performance
- Discount effectiveness
- Subscription behavior
- Seasonal purchasing patterns
- Revenue contribution by customer segments

Advanced SQL concepts used include:

- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- Subqueries
- CASE Statements
- GROUP BY & HAVING


### 4. Dashboard Development

Developed an interactive Power BI dashboard featuring:

- Sales Overview
- Customer Demographics
- Product Performance
- Revenue Analysis
- Purchase Behavior
- KPI Cards
- Interactive Filters and Slicers

---

## Key Insights

- Identified customer segments contributing the highest revenue.
- Evaluated purchasing behavior across demographic groups.
- Measured the impact of discounts on sales performance.
- Analyzed subscription trends and customer retention indicators.
- Highlighted top-performing products and categories.
- Built executive-friendly dashboards for business decision-making.
